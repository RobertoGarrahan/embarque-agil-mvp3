# 🚢 EmbarqueÁgil - MVP3 

**Sprint de Gestão Ágil de Projetos e Produtos**

## 📋 Sobre o Projeto
O **EmbarqueÁgil** é um portal web/mobile de gestão de conformidade desenhado para o setor offshore. O objetivo do produto é centralizar e otimizar o processo de validação de documentos pré-embarque (como CBSP, HUET e ASO). 

Através da plataforma, o marítimo ganha autonomia para enviar as fotos de seus certificados atualizados diretamente pelo celular, enquanto a equipe de Recursos Humanos e Logística Offshore utiliza um painel visual rápido para aprovar ou rejeitar os embarques. O foco deste MVP é reduzir o tempo gasto com a validação manual de documentos e zerar os cancelamentos de voos logísticos por pendências documentais.

## 🗂️ Estrutura do Repositório
Este repositório contém os artefatos gerados durante o planejamento e ideação do MVP, utilizando práticas de Lean Inception, Gestão de Backlog e Prototipação UI/UX.

*   📄 `canvas-url.txt`: Contém o link público para o board no **Miro** com todas as dinâmicas da Lean Inception e o MVP Canvas finalizado.
*   📄 `product-backlog.pdf`: Exportação do **Jira** contendo o Backlog do Produto priorizado, juntamente com o *Definition of Ready (DoR)* e o *Definition of Done (DoD)* aplicados ao projeto.
*   📄 `sprint-backlog.pdf`: Exportação do **Jira** detalhando o Backlog da primeira Sprint, incluindo as estimativas e os critérios de aceitação (formato BDD) das Histórias de Usuário selecionadas.
*   📁 `wireframes/`: Diretório contendo as imagens dos protótipos de baixa fidelidade desenvolvidos no **Figma** para a primeira Sprint.

## 🛠️ Ferramentas Utilizadas
*   **Miro:** Facilitação e Lean Inception.
*   **Jira Software:** Gestão Ágil (Scrum), criação de Épicos e Histórias de Usuário.
*   **Figma:** Prototipação de Interface (Wireframes Mobile-First e Desktop).
